# 关于本站

## 我是谁

全栈偏前端的工程师，常年折腾前端技术和自动化工具，喜欢把零碎的想法写成短文，也把踩过的坑记下来免得重复犯错。

## 这个博客写什么

- 架构与工程化：构建、部署、性能优化的实战记录。
- 工具与效率：脚本、自动化与习惯养成，让日常更轻松。
- 生活与阅读：夜读笔记、旅行片段以及一些个人感悟。

## 写作原则

保持诚实和可复现。文章会附带背景、操作步骤和复盘，希望读者能借鉴思路，而不是只拿到结果。

## 联系方式

欢迎在 GitHub 提 Issue 与我交流：<https://github.com/lcultra/site>。如果你也在用 VitePress 写博客，来分享你的实践吧。
